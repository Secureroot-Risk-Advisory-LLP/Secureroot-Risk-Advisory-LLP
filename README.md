SecureRoot Risk Advisory LLP is a cybersecurity consulting firm delivering industry-standard services in:

- 🔍 Vulnerability Assessment & Penetration Testing (Web, API, Mobile, Network, Infra)
- ☁️ Cloud Security Assessments (AWS, Azure, GCP)
- 🔐 Secure Code Review & Secure SDLC
- 📜 Compliance & Audit Support (ISO 27001, PCI DSS, SOC 2, GDPR, HIPAA, RBI SAR, SEBI)
